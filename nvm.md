#####查看有哪些node版本可以安装
*    `nvm ls-remote`

#####用于管理node 版本
*    `nvm`

#####查看已安装的node版本
*    `nvm list`


#####安装指定的版本，安装多版本
*    `nvm install v10.13.0`

#####切换node版本
*    `nvm use v12.3.1`

#####使用nvm use切换到指定的版本
*    `nvm use —delete-prefix 10.13.0`

#####查看当前node版本
*    `nvm current`

#####指定一个默认的node版本
*    `nvm alias default <version>`
